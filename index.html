KavoUI for GameGuardian

Key Features

  • Floating UI
  • Structure

  • Navigation
  • Welcome alert

Installation

local Kavo = loadfile("/storage/emulated/0/Download/KavoUI.lua")()

Structure Overview

local win = Kavo:CreateWindow("My Menu")
local tab = win:CreateTab("Main")
local sec = tab:CreateSection("Features")

Elements

Button

sec:AddButton("God Mode", function()
    gg.toast("Activated")
end)

Toggle

sec:AddToggle("Speed", false, function(state)
    gg.toast("Speed: "..tostring(state))
end)

Slider

sec:AddSlider("FOV", 10, 120, 60, function(v)
    gg.toast("FOV = "..v)
end)

Input

sec:AddInput("Player Name", "User", function(name)
    gg.toast("Name: "..name)
end)

Displaying the Menu

win:Show()

Welcome Alert (Optional)

Kavo:Welcome("")   -- Default message
Kavo:Welcome(nil) -- No alert
Kavo:Welcome("Your custom message")
